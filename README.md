# **JNet** 🧠

### **A free and open framework for Machine Learning**

---

### What is JNet ❓
JXNet is a free and open-source software framework for machine learning and artificial intelligence. Unlike previous frameworks, JNet is exclusively for the Java programming language, allowing the framework to focus on medium to large projects / experiments by providing a highly modular API suited for Java's object-oriented paradigm. JNet is meant for full custoization which means base classes for inheretance are provided.

JXNet is committed to democratizing and opening up machine learning and artificial intelligence to the world, so from the newest student to the most experienced scientist, we are determined to share a new technological revolution that is on the horizon.

---

### Getting Started 🚀

JXNet's interface are designed to be intuitive and modular, simmilar to PyNet.

#### JXNet StandardNet API
The StandardNet API is the main framework to build models. Most processes and intuition remain consistent across all other interfaces within JXNet with slight differences in systems or operational execution.

    JNet is a work in progress

### Core Features ⚡
At its core, JXNet is a framework full of layers and processes that require complex setup to run properly, hence, prebuilt APIs are made in order to streamline this process. Currently, JNet uses one main API to streamline all processes.
